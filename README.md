# Crowd Analytics Dashboard 

A FlowMetrics-inspired real-time crowd analytics dashboard built using **YOLOv8** and **Streamlit**.

 Features
- Live people counting using mobile IP camera
- Video-based crowd analysis (1 frame per second optimization)
- Image-based people counting
- Real-time occupancy metrics & trends
- FlowMetrics-style UI

 Tech Stack
- Python
- YOLOv8
- Streamlit
- OpenCV
